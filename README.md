# Lab 4 - Тестирование web-приложений с помощью Selenium

## Demo

![Tests](demo/demo.gif)

## Использование

Установить nodejs

Установить зависимости через `yarn`:

```
yarn
```

Запустить все тесты

```
yarn tests
```
Запустить отдельные тесты

```
yarn auth
yarn search
yarn rating
yarn sort
```
